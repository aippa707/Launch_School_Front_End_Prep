Some examples of HTML elements:

"strong" visibly changes the wrapped text to be bold. Text will have a stronger importance than the surrounding text. "em" visibly changes the wrapped text to be italicized. Demonstrates what part of the sentence is to receive emphasis. "b" visibly changes the wrapped text to be bold. Does not affect the meaning of the content as the strong element does. "i" visibly changes the wrapped text to be italicized. Does not affect the meaning of the content as the em element does. "sub" adjusts the position of the text to be lower than the text around it. Short for subscript. "sup" adjusts the position of the text to be higher than the text around it. Short for superscript.
